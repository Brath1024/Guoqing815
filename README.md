## 应用介绍

######  			面试记是一款专注于为应届毕业程序员和IT面试者提供全方位面试帮助的APP。它集成了知识社区、热点内容、面试题库、模拟练习、BUG社区、贡献题目、意见反馈等多种实用功能，旨在帮助用户更好地准备面试，增加求职成功率。

### 	三、卡路里笔记

###### 				卡路里笔记开发于2021年初，目的是服务于想减脂的人群，卡路里笔记支持数十万种食物的信息查询，可以快速知晓食物热量，并且支持统计三餐以及加餐热量摄入，根据食物摄入热量，获取推荐方案。
